## **project Description**

>>主要利用jetson nano上运行open pose，达到准确识别人体轮廓的目的，再做相关处理，简化人体动作。
再用串口通信，把处理后的数据数据传入到Arduino，通过Arduino将简化的人体动作显示到32×32点阵上，即达到人灯互动的目的，从而降低行人闯红灯的概率。
